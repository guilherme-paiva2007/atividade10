O que é HTML?

Hyper Text Markup Language. É uma linguagem de marcação voltada para o desenvolvimentos de páginas web.

Como é estruturado?

Através de tags, as quais tem diversas funcionalidades, como carregar textos, imagens, tabelas ou outras tags.

Tags HTML Básicas

Parágrafos

<p> - Blocos de texto em linha, assim como este.

Listas

<ul> - Cria uma lista não ordenada.

  • A
  • B
  • C

<ol> - Cria uma lista ordenada.

  1. D
  2. E
  3. F

<li> - Indica os elementos da lista.

  • A <- <li>
  • B <- <li>
  • C <- <li>

Tabelas

<table> - Cria uma tabela.

A A A
B B B

Dentro dela, utiliza outras tags para construir a tabela:

  • <tr> - Adiciona uma linha à tabela.
  • <td> - Adiciona um elemento com informações à tabela.
  • <th> - Adiciona um elemento de título à coluna ou linha.
<th> <th> <th>
<td> <td> <td>
<- <tr>
<- <tr>

Imagens

<img> - Insere uma imagem.

Ela requer outros atributos dentro da tag:

<img src="[localização da imagem]">

Links

<a> - Permite a ancoragem com outra página.

Link

Também requer outro atributo para funcionar:

<a href="[link de destino]">
