O que é CSS?

Cascate Style Sheets. É uma linguagem que permite a estilização de HTML.

Como é estruturado?

Utilizando seletores para atingir grupos de elementos e alterar suas propriedades (e valores).

seletor { propriedade: valor; }

O seletor indica qual grupo de elementos vai ser o alvo, sendo uma tag ou classe, ou um elemento individual, através de id.

A propriedade indica qual das propriedades vai ser modificada com o valor inserido.


Classes podem ser adicionadas à elementos no documento HTML, para serem identificadas como um grupo. Chamada por .classe.

IDs são usados para nomear um elemento em específico. Chamado por #id.


CSS funciona em cascata, fazendo com que as propriedades alteradas possam ser reutilizadas em grupos, respeitando certa hierarquia.
